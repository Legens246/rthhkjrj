# E-Z.Bio

Проект-клон сайта E-Z.Bio, позволяющий пользователям создать свою персональную био-страницу.

## Возможности

- Регистрация и авторизация пользователей
- Создание и настройка персональной страницы
- Настройка пользовательского URL
- Настройка дизайна (цвета, фон, градиенты)
- Добавление ссылок на социальные сети
- Добавление информации о себе
- Хранение профилей в базе данных MongoDB и в файловой системе

## Установка и запуск

```bash
# Установка зависимостей
npm install

# Создайте файл .env.local с следующими переменными:
# MONGODB_URI=mongodb://localhost:27017/ez-bio
# NEXTAUTH_URL=http://localhost:3000
# NEXTAUTH_SECRET=your-secret-key-change-this-in-production

# Запуск сервера разработки
npm run dev

# Сборка для продакшена
npm run build

# Запуск продакшен-версии
npm run start
```

## Структура проекта

- `/app` - Страницы приложения Next.js (App Router)
- `/components` - React компоненты
- `/lib` - Утилиты и вспомогательные функции
- `/models` - Модели MongoDB
- `/public` - Статические файлы
- `/data/profiles` - Папка для хранения профилей пользователей в JSON формате
- `/@types` - Типы TypeScript

## Технологии

- Next.js 14 - Фронтенд и API
- MongoDB - База данных
- NextAuth.js - Аутентификация
- Tailwind CSS - Стилизация
- TypeScript - Типизация

## Использование

1. Зарегистрируйтесь на сайте
2. Войдите в аккаунт
3. Настройте свой профиль в панели управления
4. Поделитесь ссылкой на ваш профиль: `e-z.bio/{ваш-никнейм}`
