# Discord Ticket Bot

Бот для техподдержки с 2 типами тикетов:

- `Краши / баги клиента`
- `Support` (жалобы, наказания, доказательства и другие вопросы)

## Что умеет

- Создает тикет-каналы в нужной категории.
- Делает тикет приватным: видит автор и поддержка.
- Для баг-репорта запрашивает:
  - Версию лаунчера
  - Версию Minecraft
  - Загрузчик (Fabric / Forge / и т.д.)
  - Описание проблемы
  - Скриншот (ссылкой, опционально)
- Для `Support` запрашивает тему, описание и доказательства.
- Закрывает тикет кнопкой или командой `/close-ticket` и удаляет канал.

## Установка

```bash
cd discord-ticket-bot
npm install
```

## Настройка

1. Скопируй `.env.example` в `.env`.
2. Заполни значения:

- `DISCORD_TOKEN` - токен бота
- `GUILD_ID` - ID сервера Discord
- `BUG_CATEGORY_ID` - ID категории для баг-тикетов
- `SUPPORT_CATEGORY_ID` - ID категории для support-тикетов
- `SUPPORT_ROLE_ID` - (опционально) роль поддержки, которая будет видеть тикеты

## Запуск

```bash
npm start
```

## Использование

1. В канале сервера выполни `/ticket-panel` (нужны права `Manage Server`).
2. Пользователи нажимают нужную кнопку и отправляют форму.
3. Бот создаст приватный канал тикета в нужной категории.
4. После решения нажми `Закрыть тикет` или используй `/close-ticket`.

## Права бота в Discord

Выдай боту минимум:

- `View Channels`
- `Send Messages`
- `Read Message History`
- `Manage Channels`
- `Use Slash Commands`
